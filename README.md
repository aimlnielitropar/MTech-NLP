## 📚 Curriculum Modules

* **Module 1: Concepts That Need Coding**
    Covers generic NLP pipelines, text processing, information retrieval, text categorization, and core evaluation metrics.
* **Module 2: Morphology & N-Grams**
    Includes inflectional morphology, regular expressions, finite automata, finite state transducers, and N-Gram language models.
* **Module 3: POS Tagging & CFG**
    Focuses on rule-based & stochastic POS, Hidden Markov Models, Context Free Grammars, Probabilistic CFG, and constituency parsing.
* **Module 4: Semantics & WSD**
    Explores phrase attachment, WordNet lexical relations, homonymy & polysemy detection, Lesk/Yarowsky algorithms, and ML-based Word Sense Disambiguation.
* **Module 5: Discourse & Coherence**
    Details reference phenomena, Hobbs' algorithm for pronouns, Centering Theory, text coherence modeling (Entity Grids), and Rhetorical Structure Theory (RST).

## 👥 Training Batches

The structural content of this curriculum is aligned to support ongoing regional educational training programs. The materials and module mappings are currently configured for deployment to **Batches 6, 7, 8, and 9**.

## 🚀 Getting Started

1. Clone this repository to your local machine.
2. Open the main portal via `index.html` in your web browser.
3. Navigate to individual topic maps located in the `/html/` directory (e.g., `nlp_module1_coding.html`).
4. Access the corresponding Jupyter Notebooks (such as `Practical1_NLP_Applications.ipynb`) for hands-on coding implementations.

## 🛠 Tech Stack

* **Frontend Design:** HTML5, Tailwind CSS, Google Fonts (Outfit & Space Mono).
* **Execution Environment:** Jupyter Notebooks (Python).
